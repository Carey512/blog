EXE 快捷操作方案

整理 Windows 下常用 EXE、脚本和快捷入口的管理方式。

把常用工具集中放到一个固定目录，例如 E:\Tools，避免散落在下载目录和桌面。

给高频工具创建桌面快捷方式，并在名称前增加数字前缀，例如 01-浏览器、02-编辑器、03-终端。

如果工具需要命令行启动，可以把目录加入 Path，或者创建一个 scripts 目录存放 .cmd/.ps1 快捷脚本。

需要管理员权限的工具单独标记，避免每次打开时才发现权限不足。

后续可以在这个文档库里记录每个 EXE 的下载地址、用途、启动参数和替代方案。
