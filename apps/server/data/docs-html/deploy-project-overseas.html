海外服务器部署项目

记录从购买海外服务器、安装 Docker 到启动 web/admin/server 的完整流程。

推荐先使用一台海外 Ubuntu 服务器，开放 22 和 80 端口，不需要域名也可以先用公网 IP 访问。

服务器准备好后安装 Docker Engine 和 Docker Compose 插件，然后从 GitHub 拉取项目代码。

进入项目目录执行 docker compose up -d --build，Nginx 会把 / 指向前台，把 /admin/ 指向后台，把 /api/ 代理到后端服务。

第一次上线后先验证 http://服务器IP/、http://服务器IP/admin/ 和 http://服务器IP/health 是否可访问。

后续有域名时，再拆成 example.com、admin.example.com、api.example.com 和 assets.example.com。
